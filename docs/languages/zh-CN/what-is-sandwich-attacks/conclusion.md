# 结论（Conclusion）

三明治攻击并非单纯的技术问题，而是与 DeFi 的公信力与可持续增长直接相关。若放任不管，用户将持续遭遇意外损失与不透明的执行成本，进而离开 DeFi、回到 CEX。

减少三明治攻击，不仅是为了用户保护，还有两点关键原因：

1. 可依赖的交易环境
在透明且公平的规则下，人人都能安心交易，这会加速 DeFi 的信任与采用。

2. 实现包含 RWA 在内的自由与去中心化市场
要迎接股票、房地产、黄金、债券等现实资产（RWA）上链的未来，必须拥有攻击面更小、更加健康的交易环境。
